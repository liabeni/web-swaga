# Домашнее задание №2 — JavaScript

В рамках задания статичная вёрстка была дополнена интерактивностью с использованием JavaScript. Страница превратилась в простое веб-приложение с динамическим поведением.

## Реализовано

- динамический рендер карточек менторов на основе данных из JavaScript
- поиск по имени и описанию
- фильтрация по выбранным параметрам
- кнопка сброса фильтров
- интерактивный блок FAQ (раскрытие и скрытие ответов)

## Структура кода

Код организован с использованием ES-модулей:

- `data/mentors.js` — данные (массив объектов)
- `ui/renderMentors.js` — рендер карточек менторов
- `ui/filters.js` — логика фильтрации и чтения состояния формы
- `ui/faq.js` — логика работы FAQ
- `main.js` — инициализация приложения и обработка событий

## Запуск проекта

Для запуска проекта рекомендуется использовать расширение **Live Server** в Visual Studio Code.  
Откройте файл `index.html` и запустите его через Live Server.
