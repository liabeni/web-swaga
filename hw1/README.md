# Домашнее задание №1 — HTML и CSS

В рамках данного задания была выполнена вёрстка веб-приложения по моему готовому макету из Figma (https://www.figma.com/design/mZR94Op60MQPMo74BsfB7k/Untitled?node-id=0-1&t=xM3H9NrC8T1RRIEh-1) с использованием HTML и CSS.

## Реализовано

- семантическая HTML-разметка основных блоков страницы
- вёрстка главного экрана, экрана поиска менторов, блока «Как это работает», FAQ и футера
- использование Flexbox и CSS Grid для построения сеток и позиционирования элементов
- адаптация под разные размеры экранов (десктоп, планшет, мобильные устройства)
- стилизация в соответствии с макетом из Figma

## Особенности реализации

- стили вынесены в отдельный CSS-файл
- используются CSS-переменные для цветов, отступов и скруглений
- классы элементов названы в соответствии с их бизнес-назначением
- структура проекта организована по папкам (`assets/styles`, `assets/images`)

## Запуск проекта

Для просмотра вёрстки рекомендуется использовать расширение **Live Server** в Visual Studio Code.  
Откройте файл `index.html` и запустите его через Live Server.
